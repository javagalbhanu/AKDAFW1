1. Enter URL & TimeOut in ./scripts/Controller.xlsx/Configuration Sheet
	By default it exeutes on Sample.html present in this framework

2. In this framework Each Sheet in ./scripts/Scenarios.xlsx represents a Script
3. Every Sheet in Scenarios.xlsx should contain 4 columns-> StepDescription,Keyword1,Keyword2,Keyword3 & Keyword4
4. StepDescription is for our reference, which is used to explain the step
5. To specify keywords in Keyword1,Keyword2,Keyword3 & Keyword4 columns refer help sheet present in Controller.xlsx
6. Specify Script Name with Status (Yes/No) in ScenarioList
7. Sheet Name in Scenarios.xlsx and  in ScenarioName Column of ScenarioList in Cotroller.xlsx should be same
8. Run KeyDriver class ; Refresh Java Project & See the result in logs folder
9. For every executing it will generate new log file
